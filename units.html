Pixeles

You can use rem values. The value is relative to the root element's font-size, which is the html element. As a result, the value will not vary depending on the depth of the element in the HTML hierarchy, and will reamin context independent.

Viewport Units

Las unidades de medida vh (viewport height) y vw (viewport width) son unidades relativas que se utilizan en CSS para especificar tamaños en relación con las dimensiones del viewport del navegador.
