Variables y Funciones en CSS

Custom Properties ( Variables CSS )

Las variables CSS permiten que un valor se almacene en un lugar y luego se haga referencia en varios otros lugares. Un beneficio adicional son los identificadores semánticos. Por ejemplo --main-text-color es más fácil de entender que #00ff00 , especialmente si este mismo color también se utiliza en otro contexto.
